# 📊 Student Grade Tracker (Java)

## 📌 Description
The Student Grade Tracker is a simple Java console application that allows users to input student names and their grades. The program calculates and displays the average, highest, and lowest scores, along with a complete summary report of all students.

## 🚀 Features
- Add multiple students with grades
- Store data using ArrayList
- Calculate average score
- Find highest and lowest scores
- Display full student report

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner (User Input)
